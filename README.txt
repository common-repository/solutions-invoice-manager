Manage invoices on your very own wordpress installation. Supports emailing, PDF viewing and Reports.

== Description ==

Solutions Invoice Manager is an all in one invoice management system. Create invoices for your customers and allow them to view them password protected on your site. There is also built in support for emailing customers with multiple templates including custom, payment reminders, services rendered, and more.

== Installation ==


1. Extract `solutions-invoice-manager.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How to backup your invoices =
Invoices can be backed up by using the Export feature that comes with WordPress. And if you need to restore them use the Import feature that comes with WordPress.
